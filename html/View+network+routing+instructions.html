SmartOS檢視網路路由指令

# netstat -rn
Routing Table: IPv4
Destination           Gateway           Flags  Ref     Use     Interface
-------------------- -------------------- ----- ----- ---------- ---------
default              211.78.255.254       UG        3        193
127.0.0.1            127.0.0.1            UH        3          8 lo0
211.78.255.0         211.78.255.31        U         3          0 net0

Routing Table: IPv6
Destination/Mask            Gateway                   Flags Ref   Use    If
--------------------------- --------------------------- ----- --- ------- -----
::1                         ::1                         UH      2       0 lo0
