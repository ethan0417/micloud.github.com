Node.js安裝與使用

使用production模式

NODE_ENV=production node app.js

必裝套件-Express

npm install express -gd
cd /usr/local/lib/node
ln -s ../node_module/express

必裝套件-EJS

npm install ejs -gd
cd /usr/local/lib/node
ln -s ../node_module/ejs
